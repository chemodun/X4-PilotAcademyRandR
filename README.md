# Pilot Academy: Ranks and Relations

This mod adds Pilot Academy functionality to the game. The main feature is to allow the player to not only train pilots using the most effective trade routes but also to use these trade operations to improve relations with various factions in the game.

## Features

- Provides a simple and effective UI to manage Academy operations.
- Makes it possible to train pilots using trade operations.
- Makes it possible to improve relations with various factions using trade operations.
- Organizes the training process using wings.
- Makes it possible to select a primary goal to either train pilots or improve relations.
- Makes its it possible to limit factions used for training and relations improvement per wing.
- Allows organizing the Academy on a non-player-controlled station with appropriate rent spending.
- After completing the relevant research, supports:
  - Up to 9 wings.
  - Training pilots up to 5-star rank.
  - Auto-hiring cadets from selected factions.
  - Auto-assigning trained pilots to existing ships outside the Academy based on selected priority.
